# 1. 项目概述

## 1.1 游戏简介

《桌面裂隙》是一款桌面常驻型地下城经营防守游戏。游戏窗口贴附在玩家电脑桌面边缘，像任务栏、桌面宠物或小型动态插件一样存在。

玩家扮演一名地下城城主。由于魔界与现实桌面之间出现了一道不稳定裂隙，玩家的地下城被压缩成一条狭长的桌面裂缝。冒险者会不定期从裂隙入口闯入，试图探索、寻宝、剿灭魔物，甚至摧毁地下城核心。

玩家需要通过建造房间、培养魔物、布置陷阱、诱骗冒险者、升级裂隙核心等方式抵抗入侵，并把冒险者转化为金币、灵魂、装备和地下城恶名。

## 1.2 一句话概念

**在你的桌面边缘，经营一座会吞掉冒险者的地下城。**

## 1.3 项目定位

# 2. 核心体验

## 2.1 桌面边缘的活体地下城

游戏不是传统全屏或普通窗口游戏，而是一条常驻在桌面边缘的地下城裂缝。玩家可以一边工作、聊天、看视频，一边看着自己的地下城自动运作。

核心感受是：

> 我的桌面底下真的住着一座地下城。

## 2.2 玩家不是勇者，而是城主

传统 RPG 中，玩家通常扮演勇者进入地下城。本作反过来，玩家经营地下城，抵抗勇者和冒险者的入侵。

冒险者不再是主角，而是地下城的威胁、资源和故事来源。

## 2.3 自动防守与轻度策略

玩家不直接控制魔物攻击，而是提前布置地下城：

- 哪个房间放在入口附近？
- 哪个房间作为诱饵？
- 哪些魔物负责前排？
- 是否要用宝箱怪诱杀盗贼？
- 如何应对牧师净化和法师清场？
- 是否故意放冒险者深入，以换取更高收益？

游戏强调“布置后的自动结果”，而不是高频点击操作。

## 2.4 挂机回来有故事

游戏的离线结算不只是资源数字，而是一份“地下城战报”。战报会总结夜间发生的入侵、魔物表现、冒险者死亡、逃跑、偷窃、房间损坏等事件。

示例：

> 昨夜共有 17 名冒险者进入地下城。  
> 其中 11 人死亡，3 人逃跑，2 人被史莱姆消化，1 名盗贼偷走了 43 金币。  
> 宝箱怪“咔咔”吞掉了一名贪婪盗贼，获得经验值。  
> 地下城恶名 +8。

# 3. 世界观设定

很久以前，地下城是独立存在的黑暗领域。某天，魔界、地下城与现实桌面之间出现了一道不稳定裂隙。

玩家作为新任地下城城主，原本应该继承一座完整魔王城。但因为裂隙坍缩，整座地下城被压缩成了一条贴在桌面边缘的狭长裂缝。

更糟糕的是，人类世界的冒险者发现了这道裂隙。他们误以为这里藏有古代宝藏，于是不断组队进入地下城。

他们想探险、寻宝、讨伐魔王、摧毁核心。

玩家的任务很简单：

**守住这条裂隙，并让冒险者明白，地下城不是给他们通关的，是用来回收他们的。**

# 4. 玩家身份与目标

## 4.1 玩家身份

玩家是地下城城主，也可以被称为：

- 裂隙之主
- 魔王代理人
- 地下城管理员
- 黑暗物业经理
- 桌面地下城城主

玩家不亲自上场战斗，而是作为管理者、建设者和策略制定者影响地下城。

## 4.2 玩家长期目标

玩家需要将一条弱小裂隙发展成强大的地下城生态系统。

长期目标包括：

1. 抵御普通冒险者入侵。
2. 建造完整地下城结构。
3. 培养强力魔物和 Boss。
4. 解锁高级陷阱与生产设施。
5. 击败知名勇者。
6. 抵抗大型讨伐事件。
7. 升级裂隙核心。
8. 建立稳定挂机收益体系。
9. 成为区域内最危险的地下城。

# 5. 游戏窗口与桌面形态

## 5.1 默认窗口形态

游戏主窗口是一条横向长条，贴附在屏幕底部。

建议尺寸：

```text
宽度：屏幕宽度的 80%–100%
高度：120px–180px
位置：屏幕底部，位于任务栏上方或可自定义贴边
```

视觉上像桌面底部被撕开一道裂隙，裂隙下方露出横版地下城剖面。

## 5.2 窗口模式

### 常驻模式

窗口长期显示在桌面边缘，地下城自动运行。

### 置顶模式

窗口显示在其他应用上方，适合观察战斗。

### 置底模式

窗口尽量不打扰用户工作。

### 收起模式

地下城缩成一条细裂缝，只显示资源、警报和小动画。

### 展开管理模式

点击裂隙或按快捷键后，展开完整管理界面，用于建造、升级、布防、查看战报。

## 5.3 桌面互动

可加入轻量桌面互动：

- 鼠标靠近时，魔物抬头看鼠标。
- 玩家可以拖拽装备投喂魔物。
- Boss 偶尔敲击屏幕边缘。
- 冒险者逃跑时撞到窗口边界。
- 入侵时出现低打扰提示。
- 收起窗口时，小怪会从裂缝里探头。

# 6. 核心循环

## 6.1 基础循环

```text
冒险者入侵
↓
地下城自动防守
↓
冒险者死亡 / 逃跑 / 掠夺 / 攻击核心
↓
玩家获得资源或承受损失
↓
升级魔物、房间、陷阱、核心
↓
地下城恶名提升
↓
吸引更强冒险者
↓
进入下一轮防守
```

## 6.2 玩家主要行为

玩家主要做五件事：

1. **建造**：扩建地下城房间。
2. **布防**：安排魔物和陷阱位置。
3. **培养**：升级和进化魔物。
4. **回收**：收集金币、灵魂、装备和材料。
5. **应对事件**：处理讨伐、偷窃、逃跑者复仇、魔物事故等事件。

# 7. 地下城系统

## 7.1 地下城结构

地下城由多个横向房间组成。

基础结构示例：

```text
裂隙入口 → 诱饵房 → 史莱姆巢穴 → 陷阱走廊 → 骷髅兵营 → 宝箱怪房 → 核心房
```

每个房间具有以下属性：

- 类型
- 等级
- 容量
- 耐久
- 可驻扎单位
- 房间效果
- 升级消耗
- 被破坏后的惩罚

## 7.2 房间类型

### 入口房

冒险者进入地下城的起点。

功能：

- 决定冒险者初始进入位置。
- 可配置第一道警戒单位。
- 可升级为伪装入口、加固入口或诱惑入口。

### 魔物巢穴

用于驻扎和恢复魔物。

示例：

- 史莱姆池
- 哥布林窝
- 骷髅兵营
- 蝙蝠洞
- 蜘蛛巢
- 宝箱怪房

### 陷阱房

用于消耗冒险者血量和状态。

示例：

- 地刺走廊
- 毒雾房
- 落石机关
- 黏液地板
- 诅咒地砖
- 火焰喷口

### 诱饵房

用于吸引冒险者深入。

示例：

- 闪光宝箱
- 假圣剑
- 金币堆
- 被困公主幻象
- 古代遗物展台

### 生产房

用于挂机产出资源。

示例：

- 地精工坊：生产建材。
- 骨粉磨坊：将尸骸转化为材料。
- 炼金池：生产毒液和药剂。
- 黑市角落：出售冒险者装备。
- 灵魂炉：将灵魂转化为魔力。

### 核心房

地下城最重要的房间。冒险者抵达核心房后会攻击裂隙核心。

核心被击破不会直接游戏失败，但会造成惩罚：

- 金币损失。
- 房间损坏。
- 魔物受伤。
- 地下城进入修复状态。
- 冒险者协会关注度上升。

核心升级路线：

| 核心类型 | 效果 |
|---|---|
| 暴君核心 | 提升魔物攻击力 |
| 深渊核心 | 强化毒雾、诅咒、持续伤害 |
| 贪婪核心 | 提升金币和掉落收益 |
| 欺诈核心 | 强化诱饵、幻象、假宝箱 |
| 繁殖核心 | 提升魔物生成和恢复速度 |

# 8. 魔物系统

## 8.1 魔物定位

魔物是地下城居民，而不是固定防御塔。它们会巡逻、睡觉、偷懒、战斗、受伤、升级、进化。

## 8.2 魔物基础属性

```text
生命值
攻击力
防御力
速度
攻击距离
士气
饥饿度
忠诚度
经验值
技能冷却
职业克制
```

## 8.3 初始魔物

### 史莱姆

定位：低成本肉盾、减速、怪海基础单位。

特点：

- 便宜。
- 数量多。
- 可以拖住敌人。
- 死亡后有概率分裂。
- 怕火焰和范围魔法。

### 哥布林

定位：杂兵、拾荒、陷阱辅助。

特点：

- 会捡掉落物。
- 会偷冒险者背包。
- 可维修陷阱。
- 胆小，遇到强敌容易逃跑。

### 骷髅兵

定位：稳定前排、中期防线。

特点：

- 不需要食物。
- 士气稳定。
- 死亡后有概率重组。
- 对圣光职业较弱。

### 蝙蝠

定位：骚扰、打断、针对后排。

特点：

- 飞行单位。
- 可越过前排。
- 能打断法师施法。
- 血量较低。

### 宝箱怪

定位：伪装、爆发、诱杀。

特点：

- 平时伪装成宝箱。
- 冒险者靠近后突然攻击。
- 对贪婪型冒险者有额外效果。
- 成本较高，但观赏性强。

## 8.4 魔物成长

魔物可以通过以下方式获得经验：

- 击败冒险者。
- 吞噬装备。
- 吸收灵魂。
- 在巢穴中训练。
- 完成特殊事件。

## 8.5 魔物进化路线示例

### 史莱姆进化

```text
普通史莱姆
→ 毒液史莱姆
→ 腐蚀史莱姆
→ 深渊史莱姆
→ 暴食史莱姆王
```

### 哥布林进化

```text
哥布林
→ 哥布林盗贼
→ 哥布林工程师
→ 哥布林陷阱师
→ 哥布林财务主管
```

### 骷髅兵进化

```text
骷髅兵
→ 骷髅盾卫
→ 骷髅弓手
→ 骷髅骑士
→ 骸骨将军
```

## 8.6 魔物个性

为增加桌面宠物感，魔物可拥有轻量个性。

示例：

| 个性 | 效果 |
|---|---|
| 贪吃 | 恢复快，但消耗食物多 |
| 胆小 | 遇到强敌容易后退 |
| 好战 | 主动追击冒险者 |
| 懒惰 | 偶尔不巡逻 |
| 忠诚 | 核心受击时优先回防 |
| 爱财 | 更容易捡金币，但可能私藏 |

# 9. 冒险者系统

## 9.1 冒险者定位

冒险者是玩家地下城的外部压力来源，也是资源来源。他们具有职业、等级、性格、目标和部分记忆。

## 9.2 冒险者目标

不同冒险者进入地下城的目标不同：

1. 探险：进入几间房后可能撤退。
2. 寻宝：优先寻找宝箱和金币。
3. 剿灭魔物：主动攻击魔物巢穴。
4. 摧毁核心：直接朝核心房推进。
5. 营救任务：被幻象或事件吸引。
6. 复仇：曾经逃跑的冒险者再次回来。

## 9.3 冒险者职业

### 战士

高生命、高防御、推进稳定，容易踩陷阱。

克制方式：

- 毒雾。
- 持续伤害。
- 怪海消耗。
- 腐蚀装备陷阱。

### 盗贼

移动快，会拆陷阱、偷金币、开宝箱。

克制方式：

- 宝箱怪。
- 反拆除陷阱。
- 蝙蝠侦查。
- 黏液减速房。

### 法师

远程高伤害，擅长范围清怪，但血量低。

克制方式：

- 蝙蝠打断。
- 沉默陷阱。
- 高速突袭单位。
- 分散式布防。

### 牧师

治疗队友、驱散诅咒，对亡灵有额外伤害。

克制方式：

- 优先突袭。
- 眩晕陷阱。
- 宝箱怪伏击。
- 切后排单位。

### 游侠

攻速快，远程攻击魔物，警觉性高。

克制方式：

- 高防御前排。
- 落石机关。
- 地形阻挡。
- 诅咒减速。

### 圣骑士

高抗性，能净化房间，常作为讨伐队核心。

克制方式：

- 物理陷阱。
- 怪海消耗。
- 腐蚀装备。
- Boss 阻挡。

### 勇者

稀有精英单位，有名字、称号、战绩和记忆。逃跑后可能变强再来，死亡后掉落独特装备。

## 9.4 冒险者性格

| 性格 | 行为倾向 |
|---|---|
| 贪婪 | 优先开宝箱 |
| 谨慎 | 会先观察或拆除陷阱 |
| 鲁莽 | 快速冲刺深入 |
| 胆小 | 队友死亡后容易逃跑 |
| 自大 | 喜欢挑战 Boss |
| 虔诚 | 诅咒抗性更高 |
| 路痴 | 可能在房间间反复徘徊 |
| 复仇心强 | 逃跑后更容易再次出现 |

# 10. 战斗与防守系统

## 10.1 战斗形式

战斗自动进行。玩家通过房间顺序、魔物配置、陷阱组合和核心技能影响结果。

## 10.2 战斗流程

```text
冒险者进入房间
↓
检测陷阱
↓
触发或拆除陷阱
↓
遭遇魔物
↓
双方自动战斗
↓
冒险者选择继续、撤退或死亡
↓
掉落资源
↓
房间结算
```

## 10.3 城主技能

玩家可以偶尔使用城主技能进行干预，但不应高频操作。

| 技能 | 效果 |
|---|---|
| 关门打狗 | 短时间封锁一个房间，使冒险者无法后退 |
| 魔力灌注 | 强化指定房间内的魔物 |
| 黑暗低语 | 降低冒险者士气，使其更容易逃跑 |
| 裂隙震荡 | 击退当前房间内的所有冒险者 |
| 临时伪装 | 将普通宝箱临时伪装成宝箱怪诱饵 |
| 紧急孵化 | 立即生成一只低级魔物 |

## 10.4 战斗结果

冒险者可能出现四种结果：

1. 死亡：掉落金币、装备、灵魂。
2. 逃跑：可能形成复仇事件。
3. 成功掠夺：偷走金币、材料或宝物。
4. 摧毁核心：造成地下城损坏和资源惩罚。

# 11. 陷阱系统

## 11.1 陷阱类型

### 地刺

基础物理伤害陷阱。

优点：便宜、稳定。  
缺点：容易被盗贼拆除。

### 毒雾

持续伤害陷阱。

优点：克制高血量单位。  
缺点：容易被牧师驱散。

### 落石

高爆发伤害陷阱。

优点：适合打集群。  
缺点：冷却长。

### 黏液地板

减速陷阱。

优点：配合怪海和远程单位。  
缺点：单独杀伤力低。

### 诅咒地砖

降低冒险者属性。

优点：适合长期消耗。  
缺点：怕圣骑士净化。

### 假宝箱

诱饵陷阱。

优点：克制贪婪冒险者和盗贼。  
缺点：对谨慎型冒险者效果较差。

## 11.2 陷阱升级示例

地刺升级：

```text
普通地刺
→ 淬毒地刺：附加中毒
→ 倒钩地刺：附加流血
→ 弹射地刺：击退敌人
```

毒雾升级：

```text
普通毒雾
→ 腐蚀毒雾：降低护甲
→ 昏睡毒雾：降低行动速度
→ 深渊毒雾：对牧师也有效
```

# 12. 资源系统

## 12.1 金币

主要通用资源。

来源：

- 冒险者掉落。
- 宝箱诱饵收益。
- 黑市售卖装备。
- 生产房产出。

用途：

- 建造房间。
- 招募魔物。
- 升级陷阱。
- 修复房间。
- 购买诱饵。

## 12.2 灵魂

高级成长资源。

来源：

- 击败冒险者。
- 击败勇者。
- 献祭装备。
- 灵魂炉产出。

用途：

- 魔物进化。
- 核心升级。
- 解锁黑暗科技。
- 激活城主技能。

## 12.3 建材

建设和维修资源。

来源：

- 地精工坊。
- 拆解装备。
- 冒险者掉落。
- 事件奖励。

用途：

- 扩建房间。
- 修复损坏。
- 强化墙体。
- 制造陷阱。

## 12.4 恶名

代表地下城的危险程度与知名度。

来源：

- 成功抵御冒险者。
- 击杀知名勇者。
- 讨伐事件胜利。
- 高死亡率地下城战报。

作用：

- 解锁更高级冒险者。
- 提高奖励品质。
- 触发大型讨伐。
- 解锁新区域和新 Boss。

恶名既是成长指标，也是难度推进器。

# 13. 事件系统

## 13.1 普通随机事件

用于增加挂机观察趣味。

示例：

- 哥布林私吞金币。
- 史莱姆繁殖过度。
- 宝箱怪吞掉钥匙。
- 地精工坊爆炸。
- 骷髅兵开始排练军乐队。
- 冒险者误入厕所房。
- 盗贼被假宝箱吓跑。
- 牧师试图净化史莱姆但失败。

## 13.2 入侵事件

普通冒险者小队进入地下城。

特点：

- 规模小。
- 难度低。
- 高频发生。
- 是主要资源来源。

## 13.3 讨伐事件

当恶名达到一定数值后触发大型讨伐。

### 新手村联合讨伐

大量低等级冒险者来袭。

特点：

- 数量多。
- 职业杂。
- 个体弱。

### 盗贼公会寻宝夜

大量盗贼进入地下城。

特点：

- 优先偷金币。
- 会拆陷阱。
- 不一定攻击核心。

### 圣光教会净化行动

牧师和圣骑士大量出现。

特点：

- 克制诅咒和亡灵。
- 会净化房间。
- 推进较慢但稳定。

### 勇者远征

命名勇者带队进入地下城。

特点：

- 强度高。
- 有特殊技能。
- 失败逃跑后可能复仇。
- 击败后有高价值奖励。

## 13.4 冒险者复仇系统

当冒险者逃离地下城时，有概率被记录为“幸存者”。幸存者可能在之后以更强状态再次出现。

复仇逻辑示例：

| 上次经历 | 下次变化 |
|---|---|
| 被毒雾击败 | 携带防毒面具，毒雾伤害降低 |
| 被宝箱怪袭击 | 会先攻击宝箱试探 |
| 被史莱姆围殴 | 携带火焰武器，对史莱姆伤害提高 |
| 抵达过核心房 | 更容易选择最短路径 |
| 队友被陷阱击杀 | 更谨慎地拆除陷阱 |

# 14. 离线收益与战报

## 14.1 设计目标

离线系统让游戏适合桌面常驻和轻度挂机，但不能让收益完全失控。

## 14.2 离线结算内容

玩家重新打开游戏时，显示“地下城夜间报告”。

报告包括：

- 入侵次数。
- 击败冒险者数量。
- 逃跑冒险者数量。
- 被偷走资源。
- 获得资源。
- 魔物成长。
- 房间损坏。
- 随机事件。
- 特殊战报。

## 14.3 战报示例

```text
地下城夜间报告

过去 8 小时内，共有 23 名冒险者进入地下城。

结果：
死亡：14 人
逃跑：5 人
成功偷窃：3 人
抵达核心房：1 人

获得：
金币 +482
灵魂 +27
破损长剑 ×4
牧师徽章 ×1

损失：
金币 -63
地刺陷阱损坏
哥布林“尖牙”轻伤

特殊事件：
宝箱怪“咔咔”吞掉了一名贪婪盗贼，获得新特性：讨厌开锁器。
```

# 16. 美术与音频风格

## 16.1 美术方向

推荐使用 **暗黑可爱像素风**。

不要做成严肃硬核暗黑风，而是强调：

- 可爱的小怪物。
- 黑色幽默。
- 微型地下城生态。
- 桌面边缘的动态小剧场。

关键词：

```text
像素风
横版剖面
小体积动画
黑暗童话
怪物生态
可爱但危险
桌面宠物感
```

## 16.2 角色表现重点

魔物需要有强识别度和小动作：

- 史莱姆弹跳。
- 哥布林偷钱。
- 骷髅兵发呆。
- 宝箱怪假装普通宝箱。
- 蝙蝠倒挂睡觉。
- Boss 打哈欠。

冒险者职业需要一眼可辨：

- 战士拿大剑。
- 盗贼戴兜帽。
- 法师拿法杖。
- 牧师有圣光图标。
- 圣骑士穿亮甲。
- 勇者有披风和独特名字牌。

## 16.3 音乐

常驻模式下音乐应低打扰。

建议：

- 默认无 BGM 或极低音量环境音。
- 地下城有轻微风声、滴水声、火把声。
- 入侵事件使用短促提示音。
- 讨伐事件切换更紧张的音乐。

## 16.4 音效

音效短、轻、可爱：

- 史莱姆弹跳声。
- 金币掉落声。
- 陷阱触发声。
- 宝箱怪咬合声。
- 冒险者卡通惨叫声。
- 核心受击警报。
- 战报结算翻页声。

必须提供“安静模式”，避免影响玩家工作。

# 20. 风险与解决方案

## 20.1 桌面窗口兼容风险

这是项目最大技术风险。

需要重点验证：

- 无边框窗口。
- 置顶窗口。
- 透明背景。
- 鼠标穿透。
- 多显示器。
- 高 DPI。
- Windows 任务栏适配。
- 全屏应用冲突。
- 用户工作时是否打扰。

解决方案：

- MVP 第一阶段先做桌面窗口技术验证。
- 不要先堆内容。
- 先确保贴边、置顶、收起、透明、点击区域逻辑可用。
- Mac 版后置，先专注 Windows。

## 20.2 自动战斗观赏性不足

如果战斗只是小人互砍，可能很快无聊。

解决方案：

- 强化陷阱触发动画。
- 强化宝箱怪、史莱姆、哥布林等记忆点单位。
- 为冒险者加入性格行为。
- 用战报文本补足叙事。
- 让不同布局产生明显战斗结果差异。

## 20.3 挂机收益失衡

挂机游戏容易出现收益过快或过慢的问题。

解决方案：

- 恶名系统作为动态难度。
- 离线收益设置上限。
- 高收益伴随高风险。
- 中后期引入讨伐压力。
- 资源消耗点持续存在。

## 20.4 内容生产压力

魔物、冒险者、事件、战报都需要大量内容。

解决方案：

- 从一开始做数据驱动结构。
- 事件文本模板化。
- 魔物和冒险者属性表格化。
- 优先做少量高辨识度内容，而不是大量低差异内容。

# 22. 项目总结

《桌面裂隙》是一款以桌面常驻窗口为核心差异点的地下城经营防守游戏。

它的关键不是做一个复杂到极致的塔防，也不是做传统地下城模拟器，而是创造一种独特体验：

> 玩家打开电脑时，桌面边缘有一条裂隙。裂隙下面有一座地下城。地下城里有魔物、有陷阱、有贪婪的冒险者、有荒诞的小事故，也有不断成长的黑暗生态。

项目最重要的设计抓手是：

1. 桌面陪伴感。
2. 自动地下城生态。
3. 冒险者入侵与回收。
4. 魔物养成与小动作。
5. 战报式轻叙事。
6. 低打扰、长期运行。
7. 黑色幽默和可爱反差。

最终目标是让玩家产生一种稳定的情感连接：

**这不是一个普通游戏窗口，而是住在我桌面底下的一座小地下城。**
