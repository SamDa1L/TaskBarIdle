# scripts 目录说明

本目录用于存放项目 GDScript 脚本。

- `autoload`：真正全局的管理器和服务。
- `desktop`：桌面窗口、贴边、收起、置顶和穿透相关逻辑。
- `dungeon`：地下城、房间、核心和布局逻辑。
- `entities`：魔物、冒险者、掉落物等实体逻辑。
- `gameplay`：入侵、战斗、陷阱、资源、成长、事件和战报逻辑。
- `resources`：自定义 `Resource` 数据类型。
- `save`：存档、读档和版本迁移。
- `ui`：界面显示和交互逻辑。
- `utils`：通用工具脚本。

脚本应优先使用信号解耦和静态类型，避免 UI、战斗和数据配置互相硬编码。
